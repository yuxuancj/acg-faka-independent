# ACG-Faka 独立版 v2.0 宝塔部署包

## 📦 使用说明

### 方式1: 自动脚本部署（推荐）

1. 下载脚本: `install_baota.sh`
2. 上传到服务器
3. 运行: `chmod +x install_baota.sh && ./install_baota.sh`
4. 按照提示输入信息

### 方式2: 手动部署

1. 下载源码
2. 在宝塔面板创建网站
3. 上传源码
4. 导入数据库
5. 设置伪静态
6. 完成安装

## 🚀 详细部署步骤

### 第一步: 创建网站

1. 登录宝塔面板
2. 点击左侧「网站」菜单
3. 点击「添加站点」
4. 填写信息：
   - **域名**: 填写您的域名（如 faka.example.com）
   - **数据库**: 选择 MySQL, 设置用户名和密码
   - **PHP版本**: 选择 8.0 或 8.1
   - **提交**: 点击「提交」

### 第二步: 上传源码

1. 在网站列表中找到刚创建的站点，点击「根目录」
2. 删除默认文件（index.html, 404.html等）
3. 点击「上传」，选择源码ZIP包
4. 上传后解压，确保文件在根目录下

### 第三步: 导入数据库

1. 在宝塔面板点击「数据库」
2. 找到刚创建的数据库，点击「管理」
3. 进入 phpMyAdmin
4. 点击「导入」，选择 `kernel/Install/install.sql`
5. 如果是从v1.0升级，还需导入 `kernel/Install/Migration_v2.0.sql`

### 第四步: 配置文件

1. 修改配置文件（如果有）
   - `config.php` 或 `config/database.php`
2. 设置数据库连接信息
3. 设置其他必要配置

### 第五步: 设置伪静态

1. 点击站点「设置」
2. 选择「伪静态」选项卡
3. 粘贴以下规则：

```nginx
location / {
    if (!-e $request_filename){
        rewrite ^(.*)$ /index.php?s=$1 last;
    }
}

location ~* (runtime|application)/.*\.php$ {
    deny all;
}

location ~ /\. {
    deny all;
}
```

4. 点击「保存」

### 第六步: 设置权限

1. 在宝塔面板中设置以下目录权限为755或777：
   - `runtime/`
   - `public/`
   - `install/` (安装完成后删除)
2. 所有者设置为 www

### 第七步: 完成安装

1. 访问您的域名 http://faka.example.com
2. 按照安装向导操作
3. 完成安装后删除 `install` 目录

## 🔧 常见问题

### 1. 打开网站显示空白/404

**解决方案**:
- 检查文件是否上传正确
- 检查伪静态规则是否正确设置
- 检查PHP版本是否为8.0+
- 检查runtime目录是否可写

### 2. 数据库连接失败

**解决方案**:
- 检查config/database.php配置
- 检查数据库用户权限
- 检查数据库是否创建

### 3. 后台登录失败

**解决方案**:
- 检查管理员账号密码是否正确
- 清除浏览器缓存
- 重置密码（使用数据库）

### 4. 上传文件失败

**解决方案**:
- 检查目录权限是否正确
- 检查php.ini中的upload_max_filesize设置
- 检查post_max_size设置
